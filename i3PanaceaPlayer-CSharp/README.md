# **C# .NET Framework (WPF)**

## **Task:**

Write a simple program that fetches content from a backend.
The program should be a desktop application with a UI on which
the content you fetch will be displayed (i.e. videoplayer).

The content you'll fetch are some media streams and their metadata that
you'll need to process and play in the app.

Requirements:

- Must be a C# .NET Framework 4.8 application.
- Use the following API endpoint: **<https://api.i3panacea.com>**
  - you can see the api specification at **<http://api.i3panacea.com/swagger>**
- The application should have a simple UI that displays the fetched content in a user-friendly way.
- The application should handle errors gracefully and provide feedback to the user if something goes wrong.
- The code should actually compile and run without errors.

When done, push your code to the repository we'll provide you with and notify us that you've completed the task.
